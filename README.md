## Installation

1.- Download source code

```
$ git clone https://github.com/SebasRod23/queue-model-simulator.git
$ cd queue-model-simulator
```

2.- Install Node.js dependencies

```
$ yarn install
```

3.- To execute the development server:

- Execute developer version

```
$ yarn start
```
